# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of OneFinance seriously. If you have found a security vulnerability, please follow the guidelines below to report it.

### How to Report

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, please report them by reaching out to the repository owners directly via their GitHub profiles: @HarshPanchal01, @JamesMeta.

### What to Include

*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   Any potential impact.

### Response

We will do our best to respond to your report as soon as possible and work with you to fix the issue.

Thank you for helping keep OneFinance safe!
